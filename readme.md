诩阆（来自于德语`schulung`)是一个工业自动化相关的公益项目，目的在于为有强烈自学能力的自动化工程师，提供一个免费学习的地方，于2018年关闭。

## 开发环境

```
ruby 2.1.5
rails 4.1.8
```

## 如何使用
1. 安装Docker
2. 运行以下命令
```bash
docker build -t xulung . 
docker run -itP xulung
```
